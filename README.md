# Structural Beam-Analysis-Tool

This project is a Python-based structural analysis tool developed to calculate:
- Maximum bending moment
- Maximum deflection
- Deflection curve visualization

The tool analyzes a simply supported beam subjected to a uniform distributed load.

## Technologies Used
- Python
- NumPy
- Matplotlib

## Features
- User input for beam length, load, modulus elasticity, and moment of inertia
- Error handling for invalid inputs
- Graphical visualization of beam delflection
- Engineering-based structural formulas

## Purpose 
This project was created to apply programming skills to real civil engineering problems and strengthen understanding of structural behavior and computational analysis.

## Example Test Case

Beam Length: 6m
Uniform Load: 10 kN/m
Modulus of Elasticity (Steel): 200 GPa
Moment of Inertia: 0.00005 m^4

This produces a symmetric deflection curve with maximum deflection occuring at midspan.
